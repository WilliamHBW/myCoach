# AI Personal Coach App (AI 个人教练)

这是一个基于 AI 的个人健身教练小程序。它可以根据你的身体状况生成专属训练计划，并分析你的运动表现。
本项目采用了 **Local-First** 策略，数据存储在本地，你需要填入自己的 AI Key (支持 OpenAI / DeepSeek / Claude) 来驱动它。

## 🚀 从零开始部署指南 (小白版)

如果你是完全没有编程基础的小白，请按照以下步骤操作，你也能拥有自己的 AI 教练！

### 第一步：准备环境
在你的电脑上安装必要的软件：
1.  **安装 Node.js**:
    *   访问 [Node.js 官网](https://nodejs.org/) 下载 "LTS" 版本并安装。
    *   安装完成后，打开终端（Terminal 或 CMD），输入 `node -v`，如果出现版本号说明安装成功。
2.  **安装微信开发者工具**:
    *   访问 [微信开发者工具官网](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html) 下载并安装。
    *   你需要有一个微信账号来登录。

### 第二步：获取代码并安装依赖
1.  **下载代码**:
    *   将本项目代码下载到你的电脑上 (如果你是直接从压缩包解压的，请跳过此步)。
2.  **安装项目依赖**:
    *   打开终端（Terminal）。
    *   输入 `cd ` (注意有空格)，然后把项目文件夹拖进终端，按回车。现在你应该进入了项目目录。
    *   输入以下命令并回车，等待安装完成（可能需要几分钟）：
        ```bash
        npm install
        ```

### 第三步：运行项目
1.  **启动编译**:
    *   在终端中继续输入：
        ```bash
        npm run dev:weapp
        ```
    *   看到 "监听文件修改中..." 或类似提示，说明编译成功了。**不要关闭这个终端窗口。**

2.  **导入微信开发者工具**:
    *   打开“微信开发者工具”。
    *   点击“导入项目” (Import Project)。
    *   **目录**: 选择本项目文件夹下的 `dist` 目录 (注意：是 `dist` 不是根目录)。
    *   **AppID**: 如果你没有申请小程序账号，点击“测试号” (Test ID) 即可。
    *   点击“确定”。

### 第四步：开始体验
现在你应该在模拟器中看到 App 界面了！

1.  **配置 AI Key (必须)**:
    *   点击底部导航栏的 **“设置”**。
    *   在 **API Key** 处输入你的大模型密钥。
        *   推荐使用 [DeepSeek](https://www.deepseek.com/) (便宜且好用) 或 OpenAI。
    *   选择对应的 **模型厂商**。
    *   点击“保存配置”。

2.  **生成计划**:
    *   点击底部 **“计划”** -> **“开始定制计划”**。
    *   回答几个关于你身体状况的问题。
    *   点击生成，见证 AI 为你定制的 4 周训练表！

3.  **导出日历**:
    *   在计划页底部，点击 **“导出到日历”**，可以将计划存为 `.ics` 文件导入手机日历。

## 🛠️ 技术栈 (开发者参考)
- **框架:** [Taro 3.x](https://taro.jd.com/) (React 18 + TypeScript)
- **状态管理:** Zustand + MMKV (持久化存储)
- **样式:** SCSS (模块化)
- **AI 交互:** 自研 LLM Client (支持多厂商 BaseURL 切换)
- **日历:** ical-generator (纯前端生成)

## 目录结构
```
src/
  ├── pages/
  │   ├── plan/          # 训练计划模块 (含问卷)
  │   ├── record/        # 运动记录模块 (含表单)
  │   └── settings/      # 设置模块
  ├── store/             # Zustand 状态管理
  ├── services/          # AI 服务层
  ├── utils/             # 工具函数 (日历生成等)
  ├── constants/         # 静态配置 (问题列表、表单字段)
  └── app.config.ts      # 全局路由配置
```

## 常见问题
- **Q: 生成计划时提示 "Network Error"?**
  - A: 请检查你的 API Key 是否正确，或者你的网络是否能访问对应的 AI 服务商 (OpenAI 国内需要代理，DeepSeek 不需要)。
- **Q: 日历导出没反应？**
  - A: 模拟器上可能无法直接打开文件，请点击“预览”生成二维码，用手机微信扫码真机体验导出功能。
